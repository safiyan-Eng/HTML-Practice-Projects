# Overview

This repository contains HTML practice projects focused on strengthening core web development fundamentals. It includes multiple beginner-level projects that demonstrate the use of semantic HTML, forms, tables, and proper page structuring.

## Projects Included

### 1. Student Registration Form
A structured HTML form that includes various input fields such as text inputs, radio buttons, checkboxes, and dropdowns. This project focuses on form handling and user input layout.

### 2. Resume / CV Webpage
A personal resume-style webpage built using HTML. It demonstrates the use of headings, lists, sections, and semantic tags to organize content in a clean and readable format.

## Key Concepts Covered

- Semantic HTML structure
- HTML forms and input elements
- Tables and lists
- Page layout organization
- Basic content structuring and readability

## Purpose

These projects were built as part of my learning journey to improve my understanding of HTML fundamentals and to practice creating well-structured and maintainable web pages.
